设备复习笔记(金鹏赞助特别版)

特别鸣谢范金鹏同志大力赞助,范老板已赞助两次😘😉👍

作者 💩 / 😋

设备复习提纲

镭雕/打标

  1. 激光特性：单色性、相干性。
    • 肉眼可见光波长：380~760mm
  2. 不同频率下的表现:
    1. 0~50Hz机械能；
    2. ＞50Hz热能；
  3. 打标机参数有：
    • 频率，速度，功率，填充，线间距，波形等；
  4. 主要影响参数有：速度，功率，填充；
  5. 谐振腔的作用：
    • ①提供反馈能量；
    • ②选择光波的方向和频率；
  6. 连续波一般用于材料焊接，脉冲一般用于材料打标；

焊接相关

  1. 焊接的形式:

    • 手工焊接:工业上不常用,统一性差
    • 自动焊接:未讲解
    • 超声焊接
    • 热熔焊接
    • 热压焊接

  2. 焊接的定义:用比被焊金属熔点低的焊料,在被焊金属不融化的基础上,焊料润湿被焊金属表面,进而使得接触面形成合金层。

  3. 焊接的三个阶段:润湿-扩散-焊点形成(焊料融化焊点不融化)

    • 润湿条件:被焊表面必须表面清洁
    • 扩散条件:合适的温度与距离
    • 焊点的形成过程:焊料和母料相互扩散，两种金属之间形成中间层(合金化)

  4. 注意事项:

    1. 焊接必要条件：
      ①焊件有良好的可焊性；
      ②焊件表面清洁无脏污；
      ③合适的助焊剂；
      ④适当温度；
      ⑤合适的焊接时间；
    2. 有铅锡焊：260±15°C；
      无铅锡焊：330±20℃；
      有铅的熔点＜无铅的熔点；
      焊料的要求:1.熔点低 2.机械强度高 3.抗氧化 4.表面张力小
    3. 助焊剂：松香(最常用)，有机助焊剂，无机助焊剂；
      助焊剂作用：有利于浸润，避免金属氧化；
    4. 烙铁：内热型和外热型；
      烙铁头形状：圆斜面，凿式，半凿式，尖锥式，圆锥式，斜面复合式，弯型；
    5. 烙铁海绵2小时一清，轻微挤一下3-4滴水为佳；

  5. 热压焊(HotBar)

    1. 热压焊特点：升温速度快，温度控制，拥有迅速的制冷性能，多段温度控制；

    2. 标准设备组成：
      ①机械部分；
      ②控制部分；
      ③监控部分；

    3. 加热片:夹持部分，支撑部分，加热部分；特点：导电性好；

    4. 加热过程：足够温度，足够时间，热传导系数大；

    5. **优点:**节能降耗，可靠性好，速度快，稳定性好，人工无法替代，外观一致性；

  6. 超声波焊接

    1. 作用：用来装配处理热塑性塑料配件及一些合成构件。例：塑料——塑料，塑料——金属；

    2. 超声波发生器将50/60Hz的电流转化为15，20，30或40KHz电流能通过换能器转化为同频次的机械运动，随后通过一组变幅杆传递到焊头，焊头传递到被焊接部位；

    3. 超声波焊接机组成：超声波发生器，换能器，变幅杆(调幅器)，焊头；

    4. 优缺点：

      1. 优点：
        ①节能环保；
        ②速度快，自动化；
        ③无需散烟散热的通风装置；
        ④成本低，效率高；
        ⑤应用广；
        ⑥焊点美观，可无缝焊接，防潮防水；
      2. 缺点：
        ①受外界影响大，对加工时的变化敏感；
        ②对材料特性敏感；
        ③大型工件密封较难完成；
        ④较难焊接半晶型树脂；
        ⑤对人听力有损害；
        ⑥破坏性焊接，不可返工；

    5. 各种组件的作用以及构成

      1. 换能器：利用电磁将电能转化为上下振幅；
      2. 变幅杆：面积越大，振幅越高；
      3. 上模材料：钛合金，钢，铝合金；
      4. 下模材料：铜，铝合金，树脂；
      5. 考虑：成本，硬度，寿命，可加工性，能量传递；

    6. 影响因素：
      ①气压；
      ②下降速度；
      ③频率；
      ④振幅；
      ⑤焊接时间；
      ⑥保压时间；

  7. 热熔设备(😋感觉不考)

    1. 两种或以上材料，热熔加压并用；

    2. 电加热型，超声波型；

    3. 加热模组：热熔头，加热黄铜，隔热板，加热棒放置处； 关键备件：铍铜；

    4. 影响：速度，温度，压力，时间；

    5. 特点：

      ①不能拆卸；

      ②链接牢固，可靠；

      ③成本低；

      ④品质稳定；

      ⑤操作简单；

      ⑥劳动强度低；

      ⑦效率较高；

      ⑧对操作员要求低；
